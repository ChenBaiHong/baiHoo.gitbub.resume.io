我的 专业介绍

JAVA 设计

Java是面向对象的语言。在java的世界中一切皆对象；之前我也学过c++;c++我也浅薄接触中。相比而言我更加钟意java， 起码在内存溢出常见的程序问题java更容易发现。java在初始化和清理，这我不会去过多考虑，清理是java内部垃圾回收机制已考虑到的问题！

SQL 设计

结构化查询语言SQL（STRUCTURED QUERY LANGUAGE）是最重要的关系数据库操作语言， 并且它的影响已经超出数据库领域，得到其他领域的重视和采用，如人工智能领域的数据检索，第四代软件开发工具中嵌入SQL的语言等。

WEB 设计

web是Java影响最深远的地方之一。网络程序设计是非常困难的，java使之得以简化。一个合格的web网页是前端和后台数据的动态交互，这里面就涉及到性能优化，在这方面我还是欠缺经验吧！

主流 框架

Spring + Struts +Hibernate，Spring + Struts +MyBatis,springMVC....。框架主要是提高工作的效率，真正要深入理解的是框架的思想；jdbc数据的访问！
